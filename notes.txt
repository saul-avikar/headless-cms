Create a user with curl:
curl 'http://localhost:3030/users/' -H 'Content-Type: application/json' --data-binary '{ "email": "feathers@example.com", "username": "feathers", "password": "secret" }'

Authenticate a user with curl:
curl 'http://localhost:3030/authentication/' -H 'Content-Type: application/json' --data-binary '{ "strategy": "local", "email": "feathers@example.com", "password": "secret" }'

Auth stratergy list:
❯◉ Username + Password (Local)
 ◯ Auth0
 ◯ Google
 ◯ Facebook
 ◯ GitHub
